# Database Configuration
# รูปแบบ: postgresql://username:password@host:port/database
# 
# หมายเหตุ:
# - ถ้า password มีอักขระพิเศษ (@, :, /, #) ให้ใช้ URL encoding

# - ถ้าใช้ localhost: postgresql://user:pass@localhost:5432/dbname
# - ถ้าใช้ remote server: postgresql://user:pass@192.168.1.100:5432/dbname
#
